Using <pre> and <code> Tags for Code Blocks

The <pre> and <code> tags are used to display code snippets in web pages.

Example code

      
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
  </head>
  <body>
    <pre>
      <code>
public class App {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}
      </code>
    </pre>
  </body>
</html>

Output

      
public class App {
    public static void main(String[] args) {
        System.out.println("Hello world");
    }
}
